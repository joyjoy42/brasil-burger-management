# Logo Brasil Burger

## Fichier du logo
Placez le logo dans ce dossier avec le nom: `logo.jpg`

Le chemin d'accès sera: `/images/logo.jpg`

## Utilisation
Le logo sera utilisé dans:
- La barre de navigation (navbar)
- La page d'accueil / Splash screen
- Le favicon (optionnel)

## Format
- JPG/JPEG (actuellement utilisé)
