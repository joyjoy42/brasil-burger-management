# Images Locales

Ce dossier contient les images locales de l'application.

## Images Requises

### Burgers
- `default-burger.png` - Image par défaut pour les burgers
- `burger-classique.jpg`
- `cheeseburger.jpg`

### Menus
- `default-menu.png` - Image par défaut pour les menus
- `menu-etudiant.png`
- `menu-poulet.png`
- `menu-tacos.png`
- `menu-famille.png`

## Note

Les images sont actuellement hébergées sur Cloudinary. Si les images Cloudinary ne sont pas accessibles, vous pouvez :
1. Ajouter des images locales dans ce dossier
2. Les images seront servies depuis `/images/`
3. Mettre à jour la base de données pour utiliser les chemins locaux
